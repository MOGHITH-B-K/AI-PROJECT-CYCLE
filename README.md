#Mathematical Foundations & Machine Learning Algorithms (Python Implementation)

This repository contains fundamental mathematical and machine learning algorithm implementations written in Python from scratch.

All programs are implemented step-by-step using core Python logic without relying on machine learning libraries such as NumPy, Scikit-learn, or TensorFlow.

The purpose of this repository is to strengthen understanding of mathematical foundations behind statistics, linear algebra, and machine learning algorithms.

* Repository Structure
🔹 Statistical Measures

MEAN.py – Calculates the arithmetic mean of a dataset

median.py – Computes the median value

mode.py – Determines the mode from user input

variance.py – Implements variance calculation

Stardard Deviation.py – Calculates standard deviation

maximum.py – Finds the maximum value in a list

minimum.py – Finds the minimum value in a list

Custom Sum.py – Computes total sum manually

🔹 Vector Operations

Vector Addition.py – Adds two vectors element-wise

Vector Subtraction.py – Performs vector subtraction

Scalar Multiplication.py – Multiplies a vector by a scalar value

🔹 Machine Learning Algorithms

Linear Regression.py – Implements simple linear regression using mathematical formula

Gradient Descent.py – Implements gradient descent optimization for linear regression
